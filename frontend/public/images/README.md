# Image Assets Directory

Please place your visual assets in this folder. The application is pre-configured to load them from these exact paths:

1. **Logo Image:**
   - **Filename:** `logo.png`
   - **Full Path:** `frontend/public/images/logo.png`
   - **Tip:** Use a PNG with a transparent background for a clean, premium integration in the header.

2. **Hero Banner Background Image:**
   - **Filename:** `hero-bg.jpg`
   - **Full Path:** `frontend/public/images/hero-bg.jpg`
   - **Tip:** Use a high-quality lifestyle image of mobile phones and premium home appliances to wow the visitor.
